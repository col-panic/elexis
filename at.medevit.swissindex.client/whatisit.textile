*Swissindex Artikel Pharma und NonPharma Downloader*  

"swissindex":http://www.refdata.ch download command line tools to fetch the current PHARMA or NONPHARMA articles. The command line tools have to be created by using @Eclipse/Export/Runnable JAR File@ .
